macro BubbleSort ( arrayAddress  ,
arraySize  ,
switchToFastModeWhileRunning   
)

Sort a 8-bit value array with bubble sort algorithm. Sorted array will be available in the same source address. Fast mode can be switched on/off while algorithm is running.

Parameters
[in,out]arrayAddressMemory address of array
[in]arraySizeArray size
[in]switchToFastModeWhileRunningIf true, fast mode will be enabled at start and disabled at end
Remarks
Registers .A, .X and .Y will be modified
Note
Use c128lib_BubbleSort in sort-global.asm
